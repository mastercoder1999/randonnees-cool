Résumé du projet : 
------------------
Notre projet est une application de données. L'objectif de celle-ci est de présenté des itinéraires possibles pour une randonnées. 
* L'application permettra de les notées et de les filtrers par distance
* L'application contiennera google map pour pin le début de l'itinéraire
* L'utilisation de swipe sera utilisé dans la page liste afin de voir les différentes itinéraires en swipant à droite et à gauche
* L'utilisation de l'accelerometre pour la vitesse de l'utilisateur

## Technologies utilisés :

Architecture : Cordova

Architecture : MVC + DAO 

Gesture : Swipe pour se promener entre les randonées en détail

Données distantes : XML


Librairie graphique externe : Google Maps


Capteur : GPS
